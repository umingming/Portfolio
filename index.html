MY SKILLS

  • Java

     Java로 기본적인 알고리즘 구현이 가능합니다. 객체 지향 프로그래밍을 이해하고 있으며 프로그램에서 필요한 클래스를 설계할 수 있습니다.

  • SQL

     오라클로 관계형 데이터베이스를 설계할 수 있습니다. SQLD를 취득하여 데이터 모델링과 SQL 기본 이론을 숙지했습니다.

  • HTML5

     HTML5로 웹페이지 레이아웃을 제작한 경험이 있습니다. Inline과 Block 속성을 이해하고 알맞게 태그를 적용할 수 있습니다.

  • CSS3

     다양한 CSS3의 속성을 이해하여 레이아웃 디자인을 적용합니다. 해상도에 따른 반응형 웹페이지를 제작할 수 있습니다.

  • JavaScript

     실제 웹페이지를 똑같이 구현하며 JavaScript 기본 문법을 공부하고 있습니다. 간단한 jquery를 사용할 수 있습니다.

  • GitHub

     형상 관리의 중요성에 대해 인지하고 있으며 GitHub을 이용해 협업을 진행한 경험이 있습니다. GitHub 서버로 웹페이지를 제작할 수 있습니다.

  • JSP

     스클립틀릿과 익스프레션을 사용해 사이트를 구현할 수 있습니다. 데이터 요청과 전송에 대해 이해하고 있으며, 알맞은 메소드를 사용합니다.

  • Servlet

     자바를 사용해 동적으로 웹 페이지를 만든 경험이 있습니다. 데이터베이스에서 필요한 데이터를 가져와 JSP에게 전달할 수 있습니다.
